# data

Single-folder curated dataset bundle for this publication repository.

## Included Files

- `metaphlan4_reversal_associations.rds`: Core microbiome reversal association object.
- `metabolic_pathway_regression_output_diet.csv`: Fecal metabolomics associations.
- `metabolic_pathway_regression_output_dietSERUMMET.csv`: Serum metabolomics associations.
- `metabolic_pathway_regression_output_dietLIPIDS.csv`: Lipid metabolomics associations.
- `metaphlan4_heatmap_input.rds`: Compact heatmap abundance input.
- `physiology_input.rds`: Compact physiology trend input table.
- `fecal_serum_lipid_microbe_metabolite_associations.xlsx`: Microbe-metabolite association workbook.
- `single_cell_degs_significant.csv`: Significant-only single-cell DEG table.
- `metabolite_classifications.csv`: Metabolite classification lookup.
- `mpa_vJan21_CHOCOPhlAnSGB_202103_SGB2GTDB.tsv`: SGB-to-GTDB mapping table.
